## Project Overview

This project analyzes the February 2024 ransomware attack on Change Healthcare.

It explains how **compromised credentials** and the **absence of multi-factor authentication** allowed attackers to gain access to critical healthcare infrastructure.  

The presentation also covers:

- **Attack chain**  
- **Operational impact** on healthcare services  
- **Monitoring failures**  
- **Incident response and mitigation**  
- **Key cybersecurity lessons**
